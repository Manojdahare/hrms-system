# HRMS System

A full-stack Human Resource Management System (HRMS) developed using Java Spring Boot and React.js.

## Features

* Employee Management
* Attendance Tracking
* Payroll Management
* Leave Management
* Authentication & Authorization
* Admin Dashboard
* Role-Based Access
* Email Notifications
* Secure REST APIs

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* Hibernate / JPA
* MySQL

### Frontend

* React.js
* JavaScript
* HTML
* CSS
* Axios

## Project Structure

* `hrms-system` → Spring Boot Backend
* `hrms-frontend` → React Frontend

## Installation

### Backend

```bash
cd hrms-system
mvn spring-boot:run
```

### Frontend

```bash
cd hrms-frontend
npm install
npm start
```
